# Accountability Atlas

A public-interest project for organizing verified evidence, documenting civic harm, and supporting lawful, accountable action.

This repository contains the project foundation, planning materials, and contributor guidance for building a transparent platform around evidence review, documentation standards, and responsible public action.

## Project focus

The initial work centers on creating a clear structure for:

- evidence documentation and source review;
- civic action templates and policy tracking;
- public-interest research workflows;
- contributor onboarding and governance.

## Repository contents

- project planning and product direction in the main documentation files;
- policy and methodology documents in the docs folder;
- issue and pull request templates under the GitHub configuration folder.

## Getting started

Review the main documentation first, then explore the contributor guidance and project roadmap before proposing changes.
